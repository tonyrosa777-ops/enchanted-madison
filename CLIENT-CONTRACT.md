### 2. Scope of Work — Pro Package ($3,000)

Optimus will design, build, and deploy a complete custom website for The Enchanted Collective. The following is included in this engagement:

#### 2.1 Pages & Core Structure

- Homepage — cinematic hero section, experience overview, social proof, and conversion CTAs
- Stays pages — The Enchanted Cottage, The Velvet Buck, Bell Tent Campsite, BYOT Campsite
- Date Night & Experiences page
- Proposals page
- Packages & Add-Ons page
- Madison, IN Local Guide
- Reviews page
- FAQ page
- Contact page
- VIP sign-up page
- Blog — architecture and 10 SEO-targeted editorial articles
- Experience Finder quiz — routes visitors to the right booking or stay

#### 2.2 Conversion Features

- Experience Finder quiz — 4-question funnel with lead capture (name, email, interest) that routes visitors to the right booking path and sends an auto-reply from angela@enchantedmadison.com
- Proposal Planner Form — 3-step animated intake form (replacing Google Form) with email notification to client
- Interactive booking calendar — demo-mode with realistic availability display (Lodgify/Acuity integration in Phase 4 once embed codes are provided)
- VIP email list capture
- Liability waiver and house rules acknowledgment — required checkboxes built into booking and experience checkout flows; guests cannot complete a reservation without signing off

#### 2.3 Technical Build

- Next.js (App Router) — deployed on Vercel
- Custom brand design system — colors, typography, spacing tokens
- Framer Motion animations sitewide — hero, scroll-driven effects, section reveals
- AI-generated photography for accommodations, experiences, and blog (fal.ai) — replaced by real photography when provided by Client
- Full SEO implementation — title tags, meta descriptions, Open Graph, schema markup
- Mobile-first responsive design — tested at 390px and scaled to desktop
- Lighthouse performance score ≥ 90 across all pages

#### 2.4 Integrations — Phase 4 (requires Client-provided embed codes)

- **Lodgify** — booking widget embedded inline on all stay pages. Client provides the Lodgify booking widget URL for each accommodation (found in Lodgify dashboard → each property → Widget/Embed). No off-domain redirect; guests book without leaving enchantedmadison.com.
- **Acuity Scheduling** — embedded for date night and proposal experience booking. The branded date/time/package selector remains. When the guest confirms, Acuity's scheduler opens in an on-page modal pre-loaded with their selected date. Client provides the Acuity scheduler URL and appointment type IDs (found in Acuity dashboard → Scheduling Page → Share).
- **Resend** — transactional email for proposal form submissions, quiz lead capture, and owner alerts. Requires verified Resend domain (enchantedmadison.com).

> **Note:** "Embed codes" means the widget URLs/snippets from each platform's dashboard — not developer API credentials. Angela can retrieve these in under 5 minutes per platform.

#### 2.5 Not Included in Pro Package

- E-commerce shop (Stripe + Printful) — available in Premium package ($5,500)
- Custom all-inclusive bundle products (Glamping + Romance Package + Add-On as one booking) — can be added as a scope addition; Client to provide bundle names, inclusions, and pricing
- Photography or video production — Client provides real assets before launch; AI placeholders are used during build
- Ongoing content writing beyond the 10 included blog articles
- Third-party tool subscriptions (Lodgify, Acuity, Resend, Vercel, fal.ai) — Client's accounts and responsibility
